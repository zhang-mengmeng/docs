搭建文档 ​

vitepress 前身是 vuepress

vitepress vuepress 静态站点生成器 Static Site Generator (SSG)

作用于项目(博客，营销页面，档案，文档)

安装 ​

npm install vitepress -D
通过npx 或者 script 去bin文件夹执行命令
